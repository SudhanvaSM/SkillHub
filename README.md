# 🚀 SkillHub – Structured Learning Platform

**Live Demo:** https://skillhub-fwd.vercel.app/
**Repository:** https://github.com/SudhanvaSM/SkillHub

---

## 📌 Overview

SkillHub is a full-stack web application designed to provide a **structured and trackable learning experience for Computer Science topics**.

It addresses a common problem among self-learners — **lack of direction and consistency** — by organizing topics into clear learning paths and enabling users to track their progress over time.

## ✨ Features

* 🔐 **JWT Authentication**

  * Secure user registration and login
  * Protected routes for authenticated access

* 📚 **Structured Learning Paths**

  * Organized CS topics into modules
  * Logical progression instead of random learning

* 📈 **Progress Tracking**

  * Track completed topics
  * Monitor learning consistency

* 🌐 **REST API Architecture**

  * Scalable backend design
  * Clear separation between frontend and backend

* 🗄️ **MongoDB Integration**

  * Efficient storage of user, course, and progress data
